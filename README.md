# Firelock Home Assistant Add-ons

This is a Home Assistant Add-on Repository.

## Add-ons

- **Wyoming Gemini Live**: Native-audio Gemini Live gateway for Home Assistant.

## How to Install

1. Go to Home Assistant > Settings > Add-ons > Add-on Store.
2. Click the 3 dots (top right) > **Repositories**.
3. Add this URL: `https://github.com/Firelock-IO/firelock-ha-addons`
4. Refresh and install the add-ons!
